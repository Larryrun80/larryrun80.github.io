Markdown为我们提供了一种非常简单的格式语法，这种语法提供了很好的可读性，并且根据这种语法书写的文本能够被自动转换为HTML格式被发布到互联网的Web服务器，被浏览器解析为丰富的呈现形式。

段落

如果在书写过程中，文本之间存在一个或多个空行，就会被认为是一个段落。

标题

用一个或多个#代表标题，#的数量决定了标题文字的大小：

引用

用>代表一个引用：

Larry 曾经说过：

Markdown 很简单易学

风格文本

你可以用 * 包围文字，使他们呈现为粗体或斜体：

这是斜体
这是粗体

你也可以用*或_嵌套显示粗体和斜体，如：

他们说粗体里可以嵌套 斜体 ，所以我来试试。

列表

你可以用* 或 - 来创建无序列表：

  • 冰激凌
  • 西瓜汁
  • 甜甜圈

如果你要创建有序列表，用数字就可以：

  1. 闹钟响了
  2. 按掉闹钟
  3. 继续睡觉

你可以使用两个空格的缩进来嵌套列表：

  1. 起床
    1. 坐起来
    2. 打哈欠
  2. 穿衣服
    • 穿衣服
    • 穿裤子
    • 穿袜子
  3. 吃早饭

代码引用：

用一个 ` （左上角，数字1旁边那个，不是单引号）包围文字在行内做代码引用

我有一个伟大的梦想

如果是多行代码引用，你需要用三个 ` 来包围文字。

超链接

用[]()来创建超链接，显示的文字写在[]中，跳转链接放在()中

他们说Larry的博客超赞的

插入图片

用 ![]() 插入图片，这个和超链的语法很像，只是在前面加个！

表格

直接上代码：

This is Table
hello world !
I am Larry

如果你觉得这些语法不够用，可以参考 Mastering Markdown
另外，你在Markdown文件中直接书写HTML，也是可以识别的。

先简单说下Sublime Text的优势：

  • 轻量级，速度快，真的超快… 在Windows上我尝试过PyCharm，速度上不说了
  • 跨语言，跨平台
  • 强大的社区支持，配置简单
    本文中，我们将尝试把Sublime Text打造成一个full stack的Python开发环境，使用一些包和特性最大化其对Python的支持。
    本文所述快捷键基于Mac OS， Windows平台可能会略有不同。

一些特性

Split Layouts | 代码分栏

这个特性允许你在同一个屏幕上分栏显示你的代码，非常适合TDD（分别显示Python代码、测试代码）和前端开发（分别显示HTML、CSS、Javascript）。
你可以用 opt + ⌘ + 2 调出双栏显示（数字1-4对应分为几栏，5为4等分屏，你也可以在View->Layout中找到他）

Chrome 风格的标签页

有助于你快速在多个文件中切换和编辑。

自动加载最后的Session

如果你不小心异常关闭了Sublime Text，放心，下次打开时他会自动加载你正在编辑的所有文件和文件夹。通常我会在Sublime Text中同时打开多个项目（速度太快没办法），如果我重新启动了电脑，他会在瞬间恢复到你启动前的样子。

Code Snippets | 代码补全

代码补全非常适合用于输入大段固定格式的代码。有了这家伙，你只需要一小段文字或关键字，系统就会帮你生成大段的预设代码。
比如说，在Python程序中，如果你输入“defs”然后按下tab键，系统就会自动帮你创建一个def 方法的代码片段。这个例子是一个预设的代码片段功能，你也可以编辑你自己的代码片段，如果你想尝试一下，可以参考这里。

  1. Setting - User 上面的 Setting - Default是全局的配置信息，里面有大多数配置项的说明和默认值，你可以根据这个文件在 Setting - User中修改你个人喜好的配置。
  2. 你也可以参考 这里 获得更多的配置说明。
  3. 如果你只想某些配置针对某些程序语言生效，编辑 Sublime Text > Preferences > Settings > More > Syntax Specific - User 文件，并且保存为 LANGUAGE.sublime-settings 。例如，如果你想保存一些Python特有的设置，保存一个Python.sublime-settings文件。我的经验是，如果你当前打开的是某个语言的文件，则系统会自动为你打开那个语言的配置文件。
  4. 你可以考虑用Dropbox同步你所有的设置，你的配置文件都可以在程序文件夹中找到。

主题

ST3为你提供了大量的主题和配色方案选择，你甚至可以自己定制，不过下载一个现成的或许更方便。你可以通过Package Control查看和安装主题。
Soda Dark Theme很流行，目前我使用的是Flatland。
配色方案上我用的是 Perv Orange， 也有人推荐Monokai Extended，你可以登陆 ColorSublime预览一下。
字体方面我使用的是14号的Ubuntu, Meslo也不错。顺便说下Mac OS 下安装字体通常的方法是下载ttf文件并双击。

扩展包推荐

活跃的社区也带来了大量优秀的扩展包，我安装了以下的一些并获益匪浅：

SideBarEnhancements

SideBarEnhancements扩展了左边栏的右键菜单功能，并且优化了一些默认的工作流。

Anaconda

Anaconda是一个终极的Python扩展包，他为ST3增加了很多IDE类的特性，包括：

  • 自动补全
  • 代码风格提示（基于PEP8），如果你不想使用这个功能，比如说你更愿意使用下文会提到的Sublime Linter，可以在 Sublime > Preferences > Package Settings > Anaconda > Settings User 中设置关闭： {"anaconda_linting": false}
  • McCabe代码复杂性检测
  • Goto Definition / 跳转到定义
  • Find Usage / 查找引用
  • 显示文档说明
    所有这些功能都在鼠标右键中得Anaconda选项下可以调出。

SublimeLinter

SublimeLinter是一个ST3 提示器框架，这个包本身不包含任何预设的提示器，你必须用Package Control安装提示包（SublimeLinter-[linter_name]）。你可以在这里找到一些官方提供的提示，实际上也存在大量第三方的提示器，查看这里。
就Python而言，我推荐使用 SublimeLinter-pyflakes 和 SublimeLinter-pep8
我也同时使用了：

  • SublimeLinter-jshint
  • SublimeLinter-pyyaml
  • SublimeLinter-csslint
  • SublimeLinter-html-tidy
  • SublimeLinter-json
    所有的提示器可能都会有依赖关系，所以安装后请阅读安装文档。

快捷键

ST3 提供了很多非常方便的快捷键：

  • 快速跳转到任何文件： ⌘+p
  • 快速调转到当前文件的某个方法： ⌘+r
  • 多处编辑：
    1. 选择一个单词，按 ⌘+d 可以选择下一个一样的单词，你可以不断按 ⌘+d，选中后可以对这些单词同时编辑
    2. 按住⌘再点击鼠标可以不断追加选择你想要的任意位置，然后同时编辑这些地方
    3. 按住⎇用鼠标拖拽可以在屏幕上选中任意多行，进行同时编辑
  • 快速跳转：
    1. ⌘+上箭头 跳转到文件头部
    2. ⌘+下箭头 跳转到文件尾部
    3. Ctrl+L 跳转到光标位置
    4. Ctrl+g 跳转到某一行
  • 切换大小写： 选中文字后按住⌘，再按K和U，将当前文字变成全大写，按住⌘，再按K，L变成全小写

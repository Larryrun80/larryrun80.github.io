Hexo提供了丰富的插件和主题，这些插件和主题基本可以帮助你实现大部分的博客需求，当然他们的安装也是相当便捷的。

插件

你可以访问 插件 访问官方的插件页面，安装基本都很简单，打开Terminal输入以下命令即可：

当然，如果你打算安装某个插件，具体安装方式还是建议参考每个插件的介绍页面，有些可能需要配置config文件

以下例举一些插件提供的功能：

  • Markdown 编辑器
  • RSS Feed
  • Site Map
  • Google Map
  • 数学表达式
  • Bootstrap支持
  • 表情
  • 增强代码表现
  • 多媒体
  • World press等系统的文章导入
  • 自动生成摘要

好吧，我承认我什么都没装…

主题

你可以访问 主题 访问官方的主题页面，和插件一样，主题的安装也很简单，基本都是clone git项目到theme文件夹：

同样建议在安装主题时访问主题的介绍页面，在成功clone主题后通常会需要编辑一下theme项目中的 _config.yml 文件。
很多主题是提供中文支持的，比如我目前使用的 PACMAN, 中文设置看 这里。这款主题也提供了非常丰富的支持系统，包括了分享、评论、甚至是Google Analytics的追踪，评论部分使用的时国内的多说，可能需要你注册一个账号并且申请一个short name( 申请页面 )。
